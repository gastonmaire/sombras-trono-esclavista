# Sombras del Trono Esclavista

## 1. Qué es este proyecto

**Sombras del Trono Esclavista** es una campaña de rol estructurada como un sistema narrativo coherente, centrado en el poder, la legitimación de la injusticia y las consecuencias morales de la estabilidad.

El proyecto está diseñado para:
- ser **consultable en mesa**
- crecer por actos sin perder coherencia
- evitar duplicación de documentos
- funcionar como archivo vivo de campaña

La **fuente canónica única** del proyecto es una **web estática navegable por hipervínculos**.

---

## 2. Fuente canónica

> **La web del proyecto es la fuente de verdad.**  
> Ningún PDF, chat o nota suelta tiene autoridad por sí mismo si contradice la web.

Todo el contenido relevante (premisa, actos, PNJ, ciudades, dungeons) vive en la web.

Los documentos externos existen solo como:
- respaldo
- material operativo de Dirección
- fuentes históricas

---

## 3. Cómo acceder a la web

### Opción A — Acceso local (sin internet)
1. Descargar la carpeta `/web` del proyecto.
2. Abrir `index.html` con cualquier navegador.
3. Navegar normalmente mediante enlaces internos.

### Opción B — Acceso online (recomendado)
La web puede publicarse como **sitio estático**, por ejemplo usando:
- GitHub Pages
- Netlify
- Cloudflare Pages

Una vez publicada, se accede mediante una URL única (ejemplo):
```
https://usuario.github.io/sombras-trono-esclavista/
```
